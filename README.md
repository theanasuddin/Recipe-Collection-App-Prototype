# Software Product and Process Management  
### Recipe Collection App — Rapid Prototype

This repository contains the rapid prototype developed for the **[Software Product and Process Management](https://opiskelijanopas.tuni.fi/en/tampere-university/curriculum/course-units/otm-4e47c4ef-57c7-406c-ab83-1d106b575186?year=2025)** course.  
The project focuses on applying product management practices, backlog creation, effort/value estimation, and iterative refinement while prototyping a Recipe Collection App using modern AI‑assisted workflows.

---

## Project Overview

The prototype explores how product and process management principles guide early‑stage software development.  
The app concept includes core features such as:

- Recipe import & personal collection  
- Ingredient scaling  
- Recipe sharing  
- Shopping list generation  
- Recipe viewing & editing  
- Tagging & search  
- Favorites & custom collections  
- Social recipe sharing & user profiles
