Follow the steps below to run the Recipe Collection App prototype locally.

----------

## Clone the Repository

```bash
git clone https://github.com/theanasuddin/Recipe-Collection-App-Prototype.git
cd Recipe-Collection-App-Prototype
cd RecipeCollectionApp

```

----------

## Install Dependencies

```bash
npm install

```

----------

## Start the Development Server

```bash
npm run dev

```

The application will start in development mode.  
Your terminal will display the local URL (typically **http://localhost:5173**).
